使用Vite创建项目

官方文档：Vite | 下一代的前端工具链

Vite是一个轻量级的，速度极快的构建工具，也是Vue3官方推荐的脚手架，使用Vite可以快速创建一个Vue3 + Typescript项目。

前置准备

你需要在Vscode的插件市场中搜索Volar并安装下面两个插件（安装插件后需要重新加载VSCode)

如果你之前安装了Vetur, 请禁用掉，并改用Volar, 它现在是 Vue 3 项目的官方推荐。

此外，你还需要在浏览器插件市场中安装Vue Dev tools

使用Vite创建项目

创建项目，在终端中输入以下命令：

npm create vite@latest

然后按照提示操作即可！

  1. Project name: 定义项目名称my-elem
  2. Select a framework: 选择Vue
  3. Select a variant: 选择TypeScript

启动项目需要执行下面的命令(当然你也可以使用yarn)

你还可以通过附加的命令行选项直接指定项目名称和你想要使用的模板，具体命令请查看官网。

编程规范-Eslint

很多人写代码有自己的风格，比如有的人喜欢在结尾加;, 但是有的人不加，这些争论还有字符串是使用单引号还是双引号，是使用空格还是tab, 正是因为每个人风格不一样，代码合并之后就会显得丑陋，如果希望团队开发者都遵循一套相同的代码规范，可以使用工具Eslint进行约束，它可以帮助你进行代码格式检测，并自动修复代码中的语法问题。

中文文档地址：https://zh-hans.eslint.org/docs/latest/use/getting-started

你可以使用该命令安装并配置 ESLint：

npm init @eslint/config

在安装的过程中，你会遇到很多次询问。

如果觉得下面的操作不够详细，可以看这篇博客：前端Vuer，请给你的项目加上 ESLint

  1. 你想怎么使用Eslint： 选择第二个
  1. 使用哪种类型的模块，这里使用Esmodules
  1. 使用什么框架
  1. 是否使用TypeScript: 选择 Yes
  2. 运行在哪种环境：选择Browser/Node, 浏览器环境和Node环境均配置
  3. 配置文件格式是什么
  1. 选择以什么工具安装上面的包：选择npm

Vite添加Eslint支持

上面的操作只是在项目中安装了Eslint, 但是还没有与Vite项目进行集成，这需要安装插件vite-plugin-eslint

npm i vite-plugin-eslint —save-dev

如果你遇到了Could not find a declaration file for module ‘vite-plugin-eslint’的报错，请参考下面的解决方案
